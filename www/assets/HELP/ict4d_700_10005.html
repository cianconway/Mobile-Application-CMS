Moodbuster Help

Symbols

You will encounter the following symbols throughout the application:

From left to right:
The Moodbuster Logo can be used to return to the home screen or exit the application. The Help page quick link leads to this help file. The Calendar quick link can be used to start the calendar, which is also available from the home screen. The same holds for the Progress quick link which leads to an overview of the exercises performed in each module. The Up button brings you one level higher in the module and is described in more detail on page 3.
